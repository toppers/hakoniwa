    WSLのインストール

    Windows版の場合はWSLをインストールいただく必要があります．
    ※注意点として，現時点(2020/3月)時点では，WSL2はご利用いただけません(WSL2とUnity間でUDP通信できないため)．

    WSLのインストール方法は色々なサイトで紹介されておりますので，Ubuntu 16.04.5 LTSをインストールください．

    以下のサイト等が参考になるかと思います．
